# Free n8n AI Agent

This is a free n8n AI agent workflow you can import into your own n8n instance.

## How to use
1. Download `ai-agent-template.json`
2. Open n8n
3. Click **Import workflow**
4. Add your own OpenAI credentials
5. Connect Google if required
6. Run the workflow
